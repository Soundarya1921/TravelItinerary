========================
   TRAVEL ITINERARY
========================

📖 Description:
A web app to plan and manage travel itineraries by adding destinations,
dates, and activities.

🚀 Features:
- Create, edit, delete itineraries
- Add destinations and schedules
- Simple and user-friendly interface

🛠️ Tech Stack:
HTML, CSS, JavaScript, React  
Node.js, Express  
MongoDB

⚙️ Run:
1. npm install
2. npm start
3. Open http://localhost:3000
